#### RC 中mysql Pod 对应的service
```bash
mysql-svc.yaml
---------------
apiVersion: v1
kind: Service
metadata:
  name: mysql   ----Service的全局唯一名称
spec:
  ports:
    - port: 3306    -----Service提供服务的端口
  selector:      ----Service对应的Pod拥有这里定义的标签
    app: mysql
```
```bash
创建services
kubectl create -f mysql-svc.yaml
查看创建services
kubectl get svc
```
