##### IP地址
```bash
它们可唯一标识IP网络中的每台设备

每台主机（计算机、网络设备、外围设备）必须具有唯一的地址

IP地址由两部分组成:
  网络ID：
     标识网络
     每个网段分配一个网络ID
   主机ID：
     标识单个主机
     由组织分配给各设备
```
