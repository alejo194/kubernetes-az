##### if 条件控制
```bash
   单条件
   if [条件测试]; then 命令序列
   fi
      
   双条件
   if [条件测试];
      then 命令序列1
      else 命令序列2
   fi
   
   多条件
   if [条件测试1];
      then 命令序列1
   elif [条件测试2];
      then 命令序列2
      else 命令序列n
   fi
```

##### for 循环控制
```bash
   for 变量名 in 值列表
   do
     命令序列
   done
```
